HUSET Jazz progressive App

Huset Jazz App Project was made based on the proposal of creating a progressive app for Huset Event House that will fullfil the needs of one persona. I have chosen to use a jazz musician as my target audience and through that I have created a app that would give only info about jazz concerts, more info about the venues and also a new event of jazz jam-sessions that any mussician could apply to build more network and have fun with other mussicians.

Role: Individual Project - survey to target group, build persona, style tile, Portotypes in paper and adobeXD, wordpress CMS and Code the app using HTML, CSS and javascript.

Alice in Wonderland

The assignment consisted of creating an animation using Javascript, HTML, and CSS. This project was a 6 member group work where we had to use a passage of a movie/animation with a given style and music by the teachers. Our given style was surrealism and a music Gods Made Love by Jimi Hendrix. So through this, we have decided to use the first scenes of the story Alice in Wonderland and create an animation with user interaction. The background and characters were made with surrealistic features and we tried to add some elements to make it more appealing to young adults once they were our target audience.

Role: Storyboard creation. Characters and background design. SVG code in title and ending.

The Mysterious Forest

The assignment was to create a story that could be future used in animation. This project had to be made following one of the styles given by the teacher. So I have created a story based on Limbo style and through that, I have made a drama and suspense animation between a nine-tails fox and a rabbit. For making this project I have worked with style tile, story creation, message, genre, storyboard, narrative curve, dramaturgic elements, character sheet/worksheet, background conception and sprite sheet - which will demonstrate a small animation of one scene.

Role: Individual Project

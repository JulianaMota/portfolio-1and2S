Artist Filho da Mãe Online Presence

This project was the final group project of 2º semester the assignment was to find an artist and improve his online presence. We chose a Portugues musician as our artist and we understood imidiatly that he had a really interesting content but need improvement in SoMe and a website to gather more fans.

Therefore our proposal was a new Artist Website working with wordpress JSON API and JavaScript to be dynamic and easly updated by the artist. We also made a stategy to improve his instagram, facebook, youtube and spotify to gather more audience.

Role: Build Report in InDesign, made wordpress CMS backend organization and JavaScript code. In documentation worked with golden circle, Competitors analysis, Costumer Journey, 5 second test, backend user manual, logo design and study.

YFU Campaign

Youth For Understanding (YFU) is a volunteer-based global non-profit company of exchange programs for youth. And the assignment was to create in group a motivational campaign to gather more youth attention. We presented our project through a communication plan based in SoMe strategy and a community website that could join more the YFU community and get more exchanges.

To create this porject we used a user centered design, working with a focus group by intreviews, 5 second test, think aloud test and Bert test to understand what would captivate our user and if our solution would be interesting. We have also worked with a promotinal video to be used in social media.

Role: Build report in InDesign, Porblem formulation, personas, digital and costumer journey, user tests, design documentation, pormotinal video and HTML and CSS code
